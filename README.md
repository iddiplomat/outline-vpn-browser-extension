# Outline VPN Proxy — Chrome Extension

Chrome-расширение для маршрутизации трафика браузера через Outline VPN (Shadowsocks) без использования приложения Outline.

## Как это работает

```
Chrome → [расширение настраивает прокси] → 127.0.0.1:1080 (sslocal) → Shadowsocks-сервер → Интернет
```

Расширение состоит из двух частей:
1. **Chrome-расширение** — управляет настройками прокси в браузере
2. **Локальный прокси** (`sslocal`) — создаёт SOCKS5-прокси, который шифрует трафик по протоколу Shadowsocks

## Установка

### 1. Установить расширение в Chrome

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `outline-vpn-plugin`

### 2. Скачать sslocal

Скачайте последний релиз **shadowsocks-rust** для Windows:
- https://github.com/shadowsocks/shadowsocks-rust/releases
- Нужен файл `shadowsocks-*-x86_64-pc-windows-msvc.zip`
- Распакуйте `sslocal.exe` в папку `outline-vpn-plugin`

Или скрипт `start-proxy.bat` предложит скачать автоматически при первом запуске.

## Использование

### Шаг 1: Запустить локальный прокси

Дважды кликните `start-proxy.bat` и передайте свой ss:// ключ:

```
start-proxy.bat "ss://YOUR_ACCESS_KEY_HERE"
```

Или через PowerShell:

```powershell
.\start-proxy.ps1 -SsKey "ss://YOUR_ACCESS_KEY_HERE"
```

По умолчанию прокси запускается на порту `1080`. Можно указать другой:

```
start-proxy.bat "ss://YOUR_KEY" 8080
```

### Шаг 2: Настроить расширение

1. Нажмите на иконку расширения в Chrome
2. Вставьте ваш `ss://` ключ Outline
3. Выберите режим:
   - **Только выбранные сайты** — укажите домены, которые нужно проксировать
   - **Весь трафик** — весь браузерный трафик пойдёт через VPN
4. Нажмите **Подключиться**

### Шаг 3: Проверить

Откройте проксируемый сайт. Если всё настроено верно, он загрузится через VPN.

## Режимы работы

| Режим | Описание |
|-------|----------|
| Только выбранные сайты | Через VPN идут только указанные домены. Остальной трафик — напрямую |
| Весь трафик | Весь трафик Chrome идёт через VPN (кроме localhost) |

### Формат списка сайтов

```
example.com
*.example.org
blocked-site.net
```

- `example.com` — проксирует точный домен и поддомены
- `*.example.org` — проксирует все поддомены

## Устранение проблем

**Сайт не открывается после подключения:**
- Убедитесь, что окно с `start-proxy.bat` открыто и работает
- Проверьте, что порт в расширении совпадает с портом в скрипте (по умолчанию 1080)

**Ошибка "Access key not set":**
- Вставьте ваш ss:// ключ в поле Access Key

**Ошибка "Invalid ss:// key format":**
- Проверьте, что ключ начинается с `ss://` и соответствует формату Outline

## Удаление

1. Отключите прокси в расширении (кнопка "Отключиться")
2. Закройте окно `start-proxy.bat`
3. Удалите расширение через `chrome://extensions/`
